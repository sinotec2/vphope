## 詳情

V2 基於 [**VuePress2**](https://vuejs.press/zh/)， 通過 [**Vite5**](https://cn.vitejs.dev/) / [**Webpack5**](https://webpack.docschina.org/) 提供通過 [**Vue3**](https://cn.vuejs.org/) 提供强大功能。

V2 完全重寫，包括:

- 所有元件都用 **Vue3 中的 Composition API** 重寫
- 所有樣式都遷移到 **Sass**

V2 也有更好的性能 🚀:

- DevServer 冷啟動時間减少了 **70%**
- 構建内存使用量减少了 **65%**
- 構建時間减少了 **75%**
- 輸出尺寸减少 **35%**
- 網頁性能提高 **60%**
